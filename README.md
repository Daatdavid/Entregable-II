#  Entregable II - Sistema de Intercambios

Este proyecto es una API REST desarrollada con **Spring Boot** para la gestión de intercambios.

## 🚀 Tecnologías utilizadas
* **Java 17**
* **Spring Boot 3.x**
* **H2 Database** (Base de datos en memoria)
* **JDBC / SQL Manual** (Capa de persistencia mediante DAO)
* **Swagger/OpenAPI** (Documentación de la API)
